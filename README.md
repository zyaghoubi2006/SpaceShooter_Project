# 🚀 Space Shooter

## 🎮 درباره بازی

این بازی یک بازی تک‌نفره است که در آن بازیکن کنترل یک سفینه فضایی را در دست می‌گیرد و باید امواج پیاپی دشمنان را نابود کند. بازی شامل ۱۰ موج (Wave) با سختی فزاینده است و بازیکن در طول بازی سکه جمع می‌کند که می‌تواند در فروشگاه بازی خرج کند.

هدف بازیکن، زنده ماندن و پاک‌سازی هر ۱۰ موج برای رسیدن به پیروزی است.

---

## 🕹️ مکانیک‌ها و امکانات پیاده‌سازی‌شده

### مکانیک اصلی گیم‌پلی
- **سیستم موج (Wave System):** ۱۰ موج با تعداد دشمن فزاینده (`5 + CurrentWave * 2`) و فاصله انتقال بین موج‌ها.
- **افزایش سختی تدریجی:** ضریب سرعت و جان دشمنان بر اساس شماره موج افزایش می‌یابد (`ApplyWaveDifficulty`).
- **اسپاون وزن‌دار دشمنان (Weighted Spawning):** نوع دشمن‌ها بسته به موج فعلی با احتمال‌های متفاوت انتخاب می‌شوند.

### انواع دشمنان
| دشمن | ویژگی | جان | امتیاز |
|------|--------|-----|--------|
| **StandardEnemy** | حرکت مستقیم به پایین | ۳۰ | ۱۰ |
| **ScoutEnemy** | حرکت موجی (سینوسی) و سریع | ۲۰ | ۲۰ |
| **ShooterEnemy** | شلیک مستقیم به بازیکن | ۵۰ | ۳۰ |
| **TerroristEnemy** | تعقیب بازیکن و انفجار هنگام برخورد | ۸۰ | ۵۰ |
| **HeavyTankEnemy** | شلیک ۸ گلوله شعاعی، جان بالا | ۲۰۰ | ۱۰۰ |

### سیستم اقتصادی و فروشگاه
- **جمع‌آوری سکه (Coin Drop):** دشمنان با احتمال متفاوت سکه‌های نقره‌ای یا طلایی می‌اندازند.
- **فروشگاه (ShopForm):** خرید و تجهیز اسکین سفینه، استایل گلوله، پس‌زمینه و آیتم مصرفی (جان اضافه).
- **ذخیره‌سازی دائمی با SQLite:** سکه‌ها، رکورد امتیاز، آیتم‌های خریداری‌شده و تنظیمات در پایگاه داده ذخیره می‌شوند.

### سیستم صوتی (Audio)
- استفاده از **NAudio** و **SoundPlayer** برای موسیقی پس‌زمینه و افکت‌های صوتی.
- تنظیمات مجزای ولوم و بی‌صدا کردن برای موسیقی و افکت‌ها (`OptionForm`).

### رابط کاربری
-  زنده **HUD** شامل جان، امتیاز، سکه و شماره موج.
- فرم‌های `OptionForm`، `ShopForm` و `AboutForm`.
- رندر با `NearestNeighbor` برای حفظ کیفیت Pixel Art.

## ▶️ راهنمای اجرا (گام‌به‌گام)

### ۱. دریافت پروژه
مخزن را کلون کنید:
git clone <repository-url>
cd SpaceShooter

### ۲. باز کردن Solution در Visual Studio
- فایل `SpaceShooter.sln` را با دابل‌کلیک باز کنید،
- یا در Visual Studio از مسیر `File > Open > Project/Solution` فایل `.sln` را انتخاب کنید.

### ۳. بازیابی پکیج‌ها (NuGet Restore)
- روی Solution راست‌کلیک کرده و **Restore NuGet Packages** را بزنید،
- یا در **Package Manager Console** دستور زیر را اجرا کنید:
powershell
Update-Package -reinstall

### ۴. بیلد کردن پروژه
- از منوی بالا `Build > Build Solution` را انتخاب کنید،
- یا کلید میان‌بر **Ctrl + Shift + B** را بزنید.

### ۵. اجرای بازی
- کلید **F5** (اجرا با دیباگ) یا **Ctrl + F5** (اجرا بدون دیباگ) را بزنید.

> ⚠️ **نکته:** مطمئن شوید پوشه `Resources` (شامل فایل‌های صوتی `.wav` و `.mp3`) در کنار فایل اجرایی خروجی کپی شده باشد. برای این کار در Visual Studio، Property فایل‌های صوتی را روی `Copy to Output Directory = Copy if newer` تنظیم کنید.
